About Project

About This Project

This project demonstrates scalable web infrastructure deployment using core AWS services.

It includes EC2 hosting, Auto Scaling Groups, Load Balancer integration, CloudWatch monitoring and SNS alerting.

The goal is to simulate a real-world highly available architecture used in production systems.

High Availability

Auto Scaling ensures resilience and uptime under varying loads.

Monitoring

CloudWatch provides real-time infrastructure metrics and alarms.

Automation

Launch templates and AMIs automate provisioning and scaling.
